# CSCI 1300 Terminal Game

A simple terminal game for the CSCI 1300 class at the University of Colorado at Boulder.  
Written by Liam Newberry & Lyle Stone.

## How to Play

To play the game on Mac:

1. Download the zip from GitHub.
2. Open a VSCode terminal in that folder.
3. In the terminal, run:
   ```bash
   g++ *.cpp -o game.out
   ./game.out

If you would like to reset the all time scores, just delete scores.txt.
